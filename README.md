# My Personal Portfolio & Blog (Built with C!) 🇳🇵

Namaste everyone! 🙏

I am a **Grade 12 Computer Science student** from Nepal. For my final project, I decided to do something different. Instead of using common backend languages like Node.js or Python, I built my website's backend using **C Programming Language**!

We learn C in Class 11 and 12, so I wanted to apply my knowledge to build something real.

## 🚀 Project Overview

This is a full-stack web application that features:

- **Frontend**: React (Vite) + Tailwind CSS for a modern UI.
- **Backend**: A custom HTTP server written in **C** (using Winsock API).
- **Database**: MySQL for storing my blog posts and projects.

## 🛠️ Tech Stack

- **C Language**: For handling HTTP requests and API logic.
- **React.js**: For the user interface.
- **MySQL**: For data storage.
- **Windows API**: For networking (Winsock).

## 📂 How to Run My Project

I have made it easy to run!

1. **Setup Database**:
    - Open MySQL Workbench.
    - Run the `database/schema.sql` file to create the `portfolio_db`.
    - *(Make sure your MySQL root user has no password, or update `backend/server.c`)*.

2. **Start Everything**:
    - Double-click the `start_all.bat` file in the main folder.
    - This will open the C server in a new window and start the React frontend.

3. **Visit the Site**:
    - Go to [http://localhost:5173](http://localhost:5173) in your browser.
